# Python Toolkit 🚀

A structured repository showcasing a series of Python mini-projects built with progressive complexity. 

The main goal of this repository is to demonstrate a solid understanding of Python's core mechanics, moving from basic syntax to object-oriented programming, file handling, and algorithmic problem-solving. Every project lives in its own folder and includes a quick guide explaining the tools and logic behind it. 

## 🛠️ Key Skills Demonstrated
* **Basics & Control Flow:** Variables, casting, loops (`for`, `while`), and conditionals (`if/elif/else`).
* **Data Structures:** Practical use of lists, dictionaries, tuples, and sets.
* **Functions & Generators:** Writing clean, modular code, and yielding dynamic data.
* **File Handling (I/O):** Reading, writing, and managing external files and directories.
* **Object-Oriented Programming (OOP):** Designing classes, creating methods, and managing state encapsulation.
* **CLI Execution:** Building interactive and user-friendly command-line applications.
